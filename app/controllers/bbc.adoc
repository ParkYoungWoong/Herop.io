== HTML

=== 문법

`HTML`의 문법은 굉장히 간단합니다. 간단한 주의사항만 지켜주시면 쉽게 `HTML`을 사용할 수 있습니다.

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>EMPTY TITLE</title>
        <link rel="stylesheet" href="css/main.css">
        <script src="js/main.js"></script>
    </head>
    <body>

    </body>
</html>
```
